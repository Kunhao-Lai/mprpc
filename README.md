# mprpc
一个基于muduo库的rpc分布式微服务框架

## 概述
1. 使用muduo开源网络库和Protobuf数据序列化和反序列化协议，实现了高效的RPC通信。
2. 使用互斥量和条件变量实现线程安全的缓冲队列，实现了异步的日志输出。
3. 使用服务器中间件ZooKeeper的服务注册和服务发现机制，实现了分布式下的服务发现功能。

## 预编译环境
1. 安装protobuf
2. 安装zookeeper
3. 安装muduo库
4. 安装CMake集成编译环境
